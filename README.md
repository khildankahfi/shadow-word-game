# Shadow Word Game - Unity

## Deskripsi
Shadow Word Game adalah aplikasi game berbasis Android yang dikembangkan menggunakan Unity dan bahasa C#. Game ini dirancang untuk memberikan pengalaman bermain sederhana dengan interaksi pengguna yang menarik.

## Fitur
- Gameplay interaktif
- Tampilan UI sederhana dan user-friendly
- Aplikasi dapat dijalankan dalam bentuk file APK
- (Opsional) Sistem skor / level (jika ada)

## Teknologi yang Digunakan
- Unity
- C#

## Platform
- Android (.apk)

2. Install di perangkat Android
